# BaseEnter

<ArgsTable of={BaseEnter} />
BaseEnter 是模板化系统中的最小单位，任何一个组件都可以进行注册到BaseEnter进行模板化

BaseEnter 组件里有一个注册表，其中包含了所有注册的组件，默认已经有 10 来种组件了，之后也可以通过动态进行注册

目前已经注册的组件：

```javascript
export const components = {
  input: Input, // element ui
  cascader: Cascader, // element ui
  label: Label, // custom
  html: HtmlText, // custom
  "date-picker": DatePicker, // element ui
  select: Select, // element ui
  option: Option, // element ui
  "radio-group": RadioGroup, // element ui
  radio: Radio, // element ui
  inline: Inline, // layout
  "row-grid": RowGrid, // layout
};
```

使用方法如下：

```html
<BaseEnter use="component name" />
```

拿 Input 组件为例子

关于组件的传参

```html
<BaseEnter
  use="input"
  placeholder="请输入内容"
  size="mini"
  :clearable="false"
  :readonly="false"
  :disabled="false"
  type="text"
  :showPassword="false"
/>
```

其中 `use` 参数是`BaseEnter`的 props，此为必选，其他参数就是原组件(element ui组件)的入参

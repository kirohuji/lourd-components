# Container 布局容器

> 用于布局的容器组件，方便快速搭建页面的基本结构：

* `<el-container>`：外层容器。当子元素中包含 `<el-header>` 或 `<el-footer>` 时，全部子元素会垂直上下排列，否则会水平左右排列。

* `<el-header>`：顶栏容器。

* `<el-aside>`：侧边栏容器。

* `<el-main>`：主要区域容器。

* `<el-footer>`：底栏容器。


## 开发经验

对于`Container`的布局思路，一般用整体的页面布局，配合vue-router等，实现一套响应式的骨架，非常适合后台管理系统的开发
